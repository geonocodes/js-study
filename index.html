HTML

DOM 구조 뼈대 구성 및 DOM객체 형성과 기본 기능 제공.

정적인 컨텐츠 작성.

  • 속성값 설정 가능.
  • onCick, onChange 등의 속성을 통한 이벤트 바인딩.
  • form 태그 : submit 버튼을 통한 액션 기능.

CSS

DOM 객체에 디자인을 적용하기 위해 사용하는 언어.

CSS선택자(DOM 객체 선택자)를 통해 객체를 선택하여 CSS를 입힘.

미리 작성된 css class를 DOM객체의 class 속성에 적용해 css를 입힘.

  • background-color, display 등 디자인 설정기능.
  • animation 효과 적용 가능.
  • 가상 클래스 혹은 해당 클래스를 통해 DOM객체에 내용 작성 가능.
